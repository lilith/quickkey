History

I started the project when I was 11 years old, and using VB6. The last build uses VB.NET 2005. I have put a significant percentage of my life into this application. It was my first real programming project, so I made a lot of mistakes, but I learned a lot in those years too. I now prefer C#, and wish I had made Quick Key a little simpler, but then maybe it would blend in with the rest of the tools.

Ways that you can help

You can lend a hand by
  • Reporting a bug to me at nathanaeljones@users.sourceforge.net.
  • Emailing me a .charset file that you have created
  • Translating Quick Key into another language.
  • Nominating a application for addition to the Keywords list
  • If you would rather dontate through PayPal, feel free.
