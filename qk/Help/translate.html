Instructions

You can translate Quick Key into another langage by downloading this plain text file. You only need to translate the text in the second column; the text in the first column must remain intact (it is only seen by the program). Please send completed files to nathanael.jones@users.sourceforge.net so that I can create a new release of Quick Key. Feel free to contact me anytime.

Thank you very much for your time and effort.
