1.   The Tray Icon and Popup Menu

 

When Quick Key starts, it places an unobtrusive icon in the system tray.  

The Tray Icon and Popup Menu is where the program’s windows and dialog boxes are controlled.  When the Quick Key Icon is clicked with the right mouse button, a menu appears.

 

On the menu, several options are displayed.

Exit – This item closes the program and removes the icon from the system tray.

About – This item displays the About Dialog Box.

Help Topics – This item starts the Quick Key Help System.

Options – This item displays the Options Dialog Box, where you can manipulate mouse settings and program keywords.

Quick Key – This item shows the Quick Key Main Window

Toolbar – This enables the additional Toolbar.

Docked – This enables the Docking System.

2.   The Quick Key Main Window

 

The Quick Key main window is responsible for the manipulation and use of foreign characters and symbols.

 

It also contains six main components

      The Character Grid

The character grid is the most often used section of the program. This is where characters are dragged, copied, and sent to and from different applications.  Each character has a popup menu from which you can select what action you wish to take with it. In the Options Dialog Box, you can also assign mouse buttons to each action. Characters in the grid can be moved around using the drag mouse button (selected in the Options Dialog Box). The font and character settings can be changed through the Toolbar.

There are three methods of transferring characters to different applications.

       The clipboard

The first is through the clipboard. A character can be copied from Quick Key and pasted into another clipboard compatible program.

       The OLE Drag- and Drop

The second is through OLE Automation and the Drag-and-Drop System. Using the drag mouse button, you can click and hold, move the cursor over the place where you wish to transfer the character to, and release the mouse button. If the program is OLE compatible, the character should successfully transfer.

       Character Sending

The third and final method is though a simulated key press. When a character is clicked with the “send” mouse button, the specified application is brought to the front and a simulated key press occurs. The application then responds to the character simulation the way it was programmed to.

      The Title Bar

This is the title bar of the Main Quick Key Window. The left mouse button can be used to move the window around the screen, and the right will display a Popup Menu.

      The Title bar Menu

 

This Menu also contains several components.

      The Hide Me Item – This item will hide the Quick Key Main Window and all of its accessory windows, but the icon will remain in the system tray.

      The Toolbar Item – This item will turn the Toolbar window on and off

      The Locked Item – This item does not allow the Quick Key Main window, the Toolbar Window, or the Docking Window to be resized or moved.

      The Chars Locked Menu – This Menu Locks the Characters in the Quick Key Main Window and does not allow them to be deleted, moved, or added.

      The Docked Item – This Menu Displays the sizable Docking Window and allows the Quick Key main Window ant The Toolbar Window to be Automatically Hid.

      The Orientation Menu – This Menu allows the user to choose the orientation of the title bar and buttons to conserve screen space, depending on the user’s needs.

      The Character Orientation Menu – This Menu allows you to choose the orientation of the order of the characters in the character grid

      The Recent Charsets Menu – This Menu displays the recent Charsets that have been loaded.

      The Docking Button

The Docking button enables the Docking Window System

      The Locking Button

                      

The Locking Button Locks the Quick Key Main Window, the Toolbar Window, and the Docking Window and does not let them be resized or moved.

      The Close Button

This button hides the Quick Key Main Window and its Accessory Windows, but does not hide the Taskbar Tray Icon.

3.   The Toolbar Window

The toolbar Window is the control center for Quick Key. Here, you can manipulate Charset files, change font settings, select the target program for sending characters, change Charset filters, and edit the character grid.

 

The Toolbar Window can be divided into five main sections

 

First, there is the Menu Bar. Most actions and options can be set from here

 

Second, there is the Command Bar. The buttons here can be used to save, open, and create new Charset files, cut, copy, paste, and delete characters, and find a new program to send characters to.

Third, there is the keyword drop-down box. This can be used to select a keyword representing the program you wish to send the character to.

Fourth, there is the font toolbar. This consists of a font name drop-down box, a font size selection box, and the font attributes buttons.

Fifth and finally, there is the status bar, which displays the information about the character under the mouse cursor.

4.   The Docking Window

The Docking Window Is used to save space on your desktop and only display the Quick Key main window and the toolbar when needed.

You can resize it to fit the available space on your desktop, and when your mouse is not over either the toolbar or the Quick Key Main window, both will disappear until the cursor is position over the docking button or the docking button is clicked. Double-clicking the Docking Window disables docking and redisplays the Quick Key Window and its accessory window, the Toolbar, if the toolbar is enabled.

5.   The Options Dialog Box

The Options Dialog Box is responsible for the Mouse Settings and the Keyword settings.

The Keyword Settings Tab

The Keyword Settings tab is where you can add, delete, find, and move program keywords (class names). A keyword is an internal nickname for each program, and is what is used for sending characters.

The Mouse Settings Tab

The Mouse Settings tab is where you assign particular functions to each mouse button. These functions will be used when clicking on the Quick Key Main Window Character Grid. Each mouse button has its own tab, which contains checkboxes listing actions the program can take when that particular mouse button is clicked.

 

On the lower section of the window there are three buttons, labeled “OK”, “Cancel”, and “Apply”, respectively. The OK button saves the settings that have been changed and closes the window. The cancel button cancels all changes and closes the window and the Apply button save changes without hiding the window.

6.   The Edit Characters Box

The edit characters box is mainly used for editing the current character set as text (found in the Tools | Edit characters as text).

It is also used for importing characters from the clipboard, other character sets, and other files.
