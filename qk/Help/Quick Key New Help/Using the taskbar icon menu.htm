Using the Taskbar Icon Pop-Up Menu

 

When Quick Key is loaded, you will see a small icon in the taskbar:

 

 

 

When this icon is right-clicked, a pop-up menu will appear

 

    The Exit item

The Exit item hides all Quick Key windows, saves all settings, and exits.

    The About Item

The About item displays the About Quick Key 5.1 Dialog Box

    The Help Topics Item

The Help Topics item displays Quick Key Help

    The Event Log Item (Debug Version Only)

The Event Log item displays the Event Log Dialog Box

    The Options Item

The Options item displays the Settings Dialog Box

    The Quick Key Item

The Quick Key item displays the Quick Key character grid

    The Toolbar Item

The Toolbar Item enables the accessory toolbar. (Only available when the Quick Key Character Grid is visible)

    The Docked Item

The Docked Item enables the docking feature. (Only available when the Quick Key Character Grid is visible)
