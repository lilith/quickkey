Included on this CD are several files, documented below.

Autorun.htm - CD Auto play

Readme.htm – Readme File

Setup.ini – Setup Information file

Setup.Exe – Quick Key Setup Boot Program – installs MSI Dependency files if necessary and then installs the Quick Key Program

Quick Key Setup.msi – Actual Setup File for the Quick Key program.

 

These files may or may not be included:

Charset Converter.exe – This is a Visual Basic 6 program for transferring text between two fonts with different character maps.

          Utilizes .cnv files, which are just text files showing which characters to replace with other characters

          Format:

                   a-b

                   b-c

                   c-d

                  

          When put through forward conversion, each a character will be replaced with a b, ever b, with a c, and every c with a d.

          This can be useful for transforming standard fonts into non-standard fonts, and vice versa.
